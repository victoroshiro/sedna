Sailor - Yacht Charter Booking HTML Template

Sailor is the first and only template especially designed for yacht charter booking companies and agencies. Sailor is a clean, modern and sleek full-screen template, using 1170px grid system with an impressively creative layout that really stands out. Thinking out of the box, it combines elements from the latest trends, such as parallax effect and infinite scrolling, but still allows users to jump between pages in order to provide the best user experience possible. It was built on information architecture’s best practices and can easily be used for any other business or portfolio website.

Installation

Sailor is a HTML template so there's no real "installation" required, just unzip the main file you've downloaded from themeforest and after unziping you have your all template files ready to be used.


Folder structure

  • css folder: contains css stylesheet files. In these files it is defined how to display HTML elements.
  • fonts folder: contains all necessary fonts.
  • images folder: contains all necessary images for the website.
  • js folder: contains javascript files. These files provide enhanced user experience and dynamic effects.

HTML structure

Head structure

The <head> element is a container for all the head elements. The head section contains elements that define the page description, keywords, and other SEO meta information. It also includes links to the stylesheets that define fonts, animations and styling of the page.

To edit the title, description or keywords of your page, open index.html or any other page for that matter and edit the text within the "content" element:

Contact form

Sailor has a working contact form. All you need to do is change the email address in contact_form_message.php file:

				$address = "joe.doe@yourdomain.com";

Javascript for form validation. This code is in js/scripts.js file.

				$('#contactform').submit(function(){
					var action = $(this).attr('action');
					$("#message").show(500,function() {
					$('#message').hide();
					$('#submit')
						.after('')
						.attr('disabled','disabled');
					
					$.post(action, { 
						name: $('#name').val(),
						email: $('#email').val(),
						comments: $('#comments').val()
					},
					function(data){
						document.getElementById('message').innerHTML = data;
						$('#message').slideDown('slow');
						$('#contactform img.loader').fadeOut('slow',function(){$(this).remove()});
						$('#submit').removeAttr('disabled'); 
					});
					
					});
					return false; 
				});
